COOKIE EXPERIMENTATION

There are two cookies on this page (one local cookie and one session cookie). One requires user input (through a prompt on the loading of the page) and the other is embedded in the page, and the user has no knowledge it is there.

The first cookie created is an example of how cookies can recognise the device and remember them. This example measures how often this site is visited by the same device. While this is different to that which this project aims to monitor, it was important to understand how cookies function.
